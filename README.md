# Identifier App v0 - 2026/1

## Example Program for Teaching Software Testing

The code for this product is part of a compiler for a language called **Silly Pascal**. It is responsible for validating whether identifiers are valid in that language.

A valid identifier contains only letters and digits. It must:

* have a length between 1 and 6 characters,
* start with a letter, and
* have the remaining characters composed only of letters or digits.

The main method of this product is:

```java
public boolean validadeIdentifier(String cadeia){...}
```

This version provides an Android graphical interface for running the application.
